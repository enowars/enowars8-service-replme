server
======
GET  /api/user/sessions
               => [ string ]
POST /api/repl { username: "string", password: "string" }
               => { replUuid: uuid }
POST /api/repl/name/:name
               => { replUuid: uuid }
WS   /api/repl/:replUuid
POST /api/repl/:replUuid/size

image
=====
-- all endpoints secured behind api-token --
POST /api/auth/register => sessionId
POST /api/auth/login    => sessionId
POST /api/term { executable: "string" }
        - spawn pty with executable under uid
        - bind pid to sessionId
WS   /api/term
POST /api/term/size
