server
======
POST /api/repl { username, password }
        - create container
            => inject api-token
        - start  container
        - POST /api/auth/register => sessionId
        - Create replId
            => { name, sessionId }
WS   /api/repl/:replId
POST /api/repl/:replId/size

image
=====
-- all endpoints secured behind api-token --
POST /api/auth/register => sessionId
POST /api/auth/login    => sessionId
POST /api/term { executable }
        - spawn pty with executable under uid
        - bind pid to sessionId
WS   /api/term/:sessionId
POST /api/term/:sessionId/size
